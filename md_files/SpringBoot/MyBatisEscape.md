# MyBatis 中的转义符

在使用MyBatis的时候，在xml的sql语句，不能直接使用大于号，小于号，而需要进行转义。常用的转义为：

| 原字符 | 转义后的字符 | 描述 |
| :------:| :------: | :------: |
| `<` | `&lt;` | 小于 |
| `<=` | `&lt;=` | 小于等于 |
| `>` | `&gt;` | 大于 |
| `>=` | `&gt;=` | 大于等于 |
| `&` | `&amp;` | 与 |
| `'` | `&apos;` | 单引号 |
| `"` | `&quot;` | 双引号 |
